# 接口基础信息

# 请求地址

  • HTTP请求地址：https://api.lnncoin.io/v1/api/，接口实际请求URL例如:https://api.lnncoin.io/v1/api/SpotBalance/GetAllBalance
  • WebSocket连接地址：wss://ws.lnncoin.io

# 频率限制

  • 目前Lnncoin只对IP进行限频，频率为100次/秒
  • 后续将根据实际情况调整每个接口的访问频率
  • 触发限频以后将返回HTTP状态码419

# HTTP请求公共参数和返回值基本结构

# 公共参数

参数名参数类型参数说明
SecretIdString密钥Id
UnixTimeInt64时间戳精确到秒
SignString签名字符串
  • 所有请求无额外header参数额
  • POST请求需要设置Content-Type:application/x-www-form-urlencoded

# HTTP请求返回值基本结构

所有应答均为json格式，基本结构如下:

{
	"Code": 0,
	"Message": "Success",
	"Data": 具体接口响应数据
}

在请求失败时，Data一般没有值

Code错误码说明
0请求处理成功
2内部错误
3参数格式、数据类型不正确
4用户数据逻辑错误
6签名错误
9内部处理超时
11服务不可用
13该接口请求过于频繁
