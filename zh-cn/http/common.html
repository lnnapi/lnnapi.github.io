# 其它公共信息模块(common)

# 获取现货交易对基础信息(SymbolInfo)

接口地址: https://common.lnncoin.io/SymbolInfo
请求方式: GET
参数列表:

参数名类型参数说明
MarketTypeString现货市场固定为Spot

返回值:

{
    "Symbols": [
        {
            "Symbol": 交易对名称(String),
            "SymbolId": 交易对ID(Int32),
            "QuoteMinNum": Quote最小精度(Float64),
            "BaseMinNum": Base最小精度(Float64),
            "IfForbidTrade": 是否禁止交易(Bool: 禁止true, 允许false)
        },
        ...
     ]
     "Coins": [
        {
            "CoinName": 币种名称(String),
            "CanWithdraw": 是否能够提现(Int8: 禁止 0, 允许 1),
            "CanDeposit": 是否能够充值(Int8: 禁止 0, 允许 1),
            "CanTransfer": 是否能够转账(Int8：禁止 0, 允许 1),
            "CoinId": 币种Id(Int32),
            "Desc": 币种描述(String),
            "WithdrawFee": 提现手续费(Float64),
            "MinDepositNum": 最小充值金额(Float64),
            "MinOperateNum": 最小操作数(Float64),
            "MinWithdrawNum": 最小提现数量(Float64)
        },
        ...
     ]

# OrderBook数据

接口地址: https://common.lnncoin.io/OrderBook
请求方式: GET
参数列表:

参数名类型参数说明
MarketTypeString现货市场固定为Spot
SymbolString交易对名称，例如BTC_USDT

响应数据解码:

  • 响应数据需要先经过base64解码，再用标准zlib解压缩得到json数据

返回值:

    {
        "Channel": "SpotOrderBook", 
        "Nonce": 消息唯一编号(Int64),
		"Symbol": 交易对名称(String),
		"Data":
		{
            "ts": 时间戳精确到毫秒(Int64)
			"Bids":
			[
				[
					价格(Float64),
					数量(Float64)	
				],
                ...
			],
			"Asks":
			[
				[
					价格(Float64),
					数量(Float64)	
				],
                ...
			]
		}
    }

# K线数据(KLine)

接口地址: https://common.lnncoin.io/KLine
请求方式: GET
参数列表:

参数名类型参数说明
MarketTypeString现货市场固定为Spot
SymbolString交易对名称, 例如BTC_USDT
KLineTypeStringK线类型, 按照不同周期可取值:1Min, 5Min, 15Min, 30Min, 1Hour, 4Hour, 24Hour, 1Week, 1Month

响应数据解码:

  • 响应数据需要先经过base64解码，再用标准zlib解压缩得到json数据

返回值:

    {  
        DataType: K线类型(String),
        Nonce: 消息唯一编号(Int64),
        Symbol: 交易对名称(String),
        IsUpdate: 固定为false,
        Data: [
            [
              时间戳(Int64),
              交易次数(Int32),
              交易量(Float64),
              交易额(Float64),
              开盘价(Float64),
              收盘价(Float64),
              最高价(Float64),
              最低价(Float64)
            ],
            ...
        ]
    }
