# 准备

首先需要准备好由自己掌握私钥的Ed25519密钥对，或前往 https://lnncoin.io/zh-cn/apikey/ed25519open in new window 进行密钥对生成。 之后请登录lnn平台 https://lnncoin.io/zh-cn/apikeyopen in new window 进行SecretId生成和公钥绑定。

名词说明
SecretId密钥Id: 平台生成，用于标识API实例 (等同于API Key)
公钥用户提供，平台用于验证签名
私钥用于对请求信息进行签名，注意用户永远不对平台或其他任何人展示自己的私钥
