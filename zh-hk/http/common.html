# 獲取現貨交易對基礎信息(SymbolInfo)

接口地址: https://common.lnncoin.io/SymbolInfo
請求方式: GET
參數列表:

參數名類型參數説明
MarketTypeString現貨市場固定為Spot

返回值:

{
    "Symbols": [
        {
            "Symbol": 交易對名稱(String),
            "SymbolId": 交易對ID(Int32),
            "QuoteMinNum": Quote最小精度(Float64),
            "BaseMinNum": Base最小精度(Float64),
            "IfForbidTrade": 是否禁止交易(Bool: 禁止true, 允許false)
        },
        ...
     ]
     "Coins": [
        {
            "CoinName": 幣種名稱(String),
            "CanWithdraw": 是否能夠提現(Int8: 禁止 0, 允許 1),
            "CanDeposit": 是否能夠充值(Int8: 禁止 0, 允許 1),
            "CanTransfer": 是否能夠轉賬(Int8：禁止 0, 允許 1),
            "CoinId": 幣種Id(Int32),
            "Desc": 幣種描述(String),
            "WithdrawFee": 提現手續費(Float64),
            "MinDepositNum": 最小充值金額(Float64),
            "MinOperateNum": 最小操作數(Float64),
            "MinWithdrawNum": 最小提現數量(Float64)
        },
        ...
     ]

# OrderBook數據

接口地址: https://common.lnncoin.io/OrderBook
請求方式: GET
參數列表:

參數名類型參數説明
MarketTypeString現貨市場固定為Spot
SymbolString交易對名稱，例如BTC_USDT

響應數據解碼:

  • 響應數據需要先經過base64解碼，再用標準zlib解壓縮得到json數據

返回值:

    {
        "Channel": "SpotOrderBook", 
        "Nonce": 消息唯一編號(Int64),
		"Symbol": 交易對名稱(String),
		"Data":
		{
            "ts": 時間戳精確到毫秒(Int64)
			"Bids":
			[
				[
					價格(Float64),
					數量(Float64)	
				],
                ...
			],
			"Asks":
			[
				[
					價格(Float64),
					數量(Float64)	
				],
                ...
			]
		}
    }

# K線數據(KLine)

接口地址: https://common.lnncoin.io/KLine
請求方式: GET
參數列表:

參數名類型參數説明
MarketTypeString現貨市場固定為Spot
SymbolString交易對名稱, 例如BTC_USDT
KLineTypeStringK線類型, 按照不同周期可取值:1Min, 5Min, 15Min, 30Min, 1Hour, 4Hour, 24Hour, 1Week, 1Month

響應數據解碼:

  • 響應數據需要先經過base64解碼，再用標準zlib解壓縮得到json數據

返回值:

    {  
        DataType: K線類型(String),
        Nonce: 消息唯一編號(Int64),
        Symbol: 交易對名稱(String),
        IsUpdate: 固定為false,
        Data: [
            [
              時間戳(Int64),
              交易次數(Int32),
              交易量(Float64),
              交易額(Float64),
              開盤價(Float64),
              收盤價(Float64),
              最高價(Float64),
              最低價(Float64)
            ],
            ...
        ]
    }
