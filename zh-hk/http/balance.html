# 獲取用戶所有資金信息(GetAllBalance)

接口: /SpotBalance/GetAllBalance
請求方式: GET
參數列表: 无

返回值:

[
	{
		"Coin": 幣種名稱(String),
		"Available": 賬戶可用數目(Float64),
		"Frozen": 凍結數目(Float64)
	},
	...
]

# 獲取指定幣種的資金信息(GetBalance)

接口: /SpotBalance/GetBalance
請求方式: GET
參數列表:

參數名類型參數説明
CoinString幣種名稱，例如 BTC

返回值:

	{
		"Coin": 幣種名稱(String),
		"Available": 賬戶可用數目(Float64),
		"Frozen": 凍結數目(Float64)
	}
