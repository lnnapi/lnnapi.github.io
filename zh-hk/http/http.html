# 接口基礎信息

# 請求地址

  • HTTP請求地址：https://api.lnncoin.io/v1/api/，接口實際請求URL例如:https://api.lnncoin.io/v1/api/SpotBalance/GetAllBalance
  • WebSocket連接地址：wss://ws.lnncoin.io

# 頻率限製

  • 目前Lnncoin只對IP進行限頻，頻率為100次/秒
  • 後續將根據實際情況調整每個接口的訪問頻率
  • 觸發限頻以後將返回HTTP狀態碼419

# HTTP請求公共參數和返回值基本結構

# 公共參數

參數名參數類型參數説明
SecretIdString密鑰Id
UnixTimeInt64時間戳精確到秒
SignString簽名字符串
  • 所有請求無額外header參數額
  • POST請求需要設置Content-Type:application/x-www-form-urlencoded

# HTTP請求返回值基本結構

所有應答均為json格式，基本結構如下:

{
	"Code": 0,
	"Message": "Success",
	"Data": 具體接口響應數據
}

在請求失敗時，Data一般沒有值

Code錯誤碼說明
0請求處理成功
2內部錯誤
3參數格式、數據類型不正確
4用戶數據邏輯錯誤
6簽名錯誤
9內部處理超時
11服務不可用
13該接口請求過於頻繁
