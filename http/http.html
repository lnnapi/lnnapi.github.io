# Interface Basic Info

# Request Address

  • HTTP request address: https://api.lnncoin.io/v1/api/, for instance the complete request URL could be: https://api.lnncoin.io/v1/api/SpotBalance/GetAllBalance
  • WebSocket request address: wss://ws.lnncoin.io

# Frequency Rate Limitation

  • For now, Lnncoin only limits access rate by IP，it's 100 times per second
  • In the future, We will activate rate limitation precisely on each interface according to online condition
  • If you exceed the limitation, you would receive the HTTP Code 419

# HTTP Common Parameter and Response

# Common Parameter

ParameterTypeExplain
SecretIdStringThe secret Id
UnixTimeInt64Timestamp, is accurate to second
SignStringSignature of the request
  • HTTP request needs no extra header data
  • If it's a POST request，please set Content-Type:application/x-www-form-urlencoded

# HTTP Response Structure

Response has json data structure:

{
	"Code": 0,
	"Message": "Success",
	"Data": Corresponding data detail
}

The Data would be empty if the request is fail.

Error CodeExplain
0Successful
2Inner error
3The format or type of the parameter is illegal
4The Logic of provided data is wrong
6Signature is error
9The server's process is over time
11The server is unavailable
13The interface is requested too frequently
