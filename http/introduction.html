# Preparation

First of all, you should prepare a safe Ed25519 key pair whose private key is only controlled by yourself. If you don't have any, you can generate through https://lnncoin.io/en-us/apikey/ed25519open in new window . Then you can generate your SecretId and bind your public key through https://lnncoin.io/en-us/apikeyopen in new window.

NameExplain
SecretIdGenerated by Lnncoin, do indicate which API instance is requesting
Public keyProvided by user, by which Lnncoin can verify the request
Private keyUser signs the request with the corresponding private key. Pay attention not to disclosing the private key to anyone including Lnncoin
