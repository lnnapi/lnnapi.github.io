# Common

# SymbolInfo(Retrieve spot symbol pair basic info)

Interface address: https://common.lnncoin.io/SymbolInfo
Method: GET
Parameter:

ParameterTypeExplain
MarketTypeStringSpot

Response:

{
    "Symbols": [
        {
            "Symbol": Symbol name(String),
            "SymbolId": Symbol Id(Int32),
            "QuoteMinNum": Quote minimum precision(Float64),
            "BaseMinNum": Base minimum precision(Float64),
            "IfForbidTrade": Indicate whether this symobl pair is forbidden to trade(Bool)
        },
        ...
     ]
     "Coins": [
        {
            "CoinName": Coin name(String),
            "CanWithdraw": (Int8: Forbidden 0, allowed 1),
            "CanDeposit": (Int8: Forbidden 0, allowed 1),
            "CanTransfer": (Int8：Forbidden 0, allowed 1), 
            "CoinId": Coin Id(Int32),
            "Desc": Coin description(String),
            "WithdrawFee": (Float64),
            "MinDepositNum": Minimum deposit amount(Float64),
            "MinOperateNum": Minimum operation amount(Float64),
            "MinWithdrawNum": Minimum withdrawal amount(Float64)
        },
        ...
     ]

# OrderBook

Interface address: https://common.lnncoin.io/OrderBook
Method: GET
Parameter:

ParameterTypeExpain
MarketTypeStringSpot
SymbolStringSymbol name, eg. BTC_USDT

Decode the response data:

  • Response data should be decoded by base64 first, then decompressed by standard zlib to get final json data

Response:

    {
        "Channel": "SpotOrderBook", 
        "Nonce": Unique Id(Int64),
		"Symbol": Symobl name(String),
		"Data":
		{
            "ts": Timestamp accurate to millisecond (Int64)
			"Bids":
			[
				[
					Price(Float64),
					Amount(Float64)	
				],
                ...
			],
			"Asks":
			[
				[
					Price(Float64),
					Amount(Float64)	
				],
                ...
			]
		}
    }

# KLine

Interface address: https://common.lnncoin.io/KLine
Method: GET
Parameter:

ParameterTypeExplain
MarketTypeStringSpot
SymbolStringSymbol name, eg. BTC_USDT
KLineTypeStringType of KLine, can be one of: 1Min, 5Min, 15Min, 30Min, 1Hour, 4Hour, 24Hour, 1Week, 1Month

Decode the response data:

  • Response data should be decoded by base64 first, then decompressed by standard zlib to get final json data

Response:

    {  
        DataType: (String),
        Nonce: Unique Id(Int64),
        Symbol: Symobl name(String),
        IsUpdate: false,
        Data: [
            [
              Timestamp(Int64),
              Trade count(Int32),
              Amount(Float64),
              Volume(Float64),
              Open(Float64),
              Close(Float64),
              High(Float64),
              Low(Float64)
            ],
            ...
        ]
    }
